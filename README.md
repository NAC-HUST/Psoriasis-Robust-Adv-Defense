## 项目简介

本项目旨在研究一种**面向银屑病医学影像诊断的对抗攻防协同鲁棒性增强方法**。针对深度学习模型在银屑病辅助诊断中面临的“对抗脆弱性”问题（即微小扰动可能导致灾难性误诊），本项目通过攻击策略与防御机制的动态交互与协同演化，提升模型对扰动的内在鲁棒性，进而提高银屑病医学影像诊断的准确性与可靠性。

### 研究背景
银屑病作为一种复杂的慢性炎症性皮肤病，临床表型多样且易与其他皮肤病混淆。虽然深度学习技术在辅助诊断中展现出巨大潜力，但现有模型对输入空间的微小扰动极度敏感，存在严重的安全隐患。本项目致力于解决这一核心瓶颈，推动智慧医疗系统的可信落地。

## 核心研究内容

本项目主要包含以下三个核心模块：

### 1. 面向医学影像的定向对抗攻击方法
- 研究生成人眼不可察觉且具有高迁移性的对抗扰动方法。
- 设计多目标攻击策略，在保证扰动微小的同时最大化分类器误判率。
- 通过分析攻击成功区域，定位模型决策中的脆弱环节，挖掘对诊断敏感的关键区域。

### 2. 基于对抗样本反馈的鲁棒分类器优化
- 提取模型在医学影像中的决策脆弱区域，作为空间注意力引导信号。
- 构建融合区域感知的对抗训练与特征一致性正则化机制。
- 在关键诊断区域增强特征稳定性，在非敏感区域适度松弛约束。

### 3. 攻击 - 防御协同演化的联合优化框架
- 设计动态交互的闭环优化系统，使攻击模块与防御模块相互驱动、协同演化。
- 引入多目标攻防机制，驱动攻击器与防御器在训练中协同演化。
- 最终形成既能精准识别病灶、又对扰动具有强内在鲁棒性的诊断系统。
