关键指标说明 ​

成功率 (Success Rate)

模型在对抗样本上做出错误预测的比例。值越高攻击效果越好。

SR=错误预测数总样本数×100%
稀疏性 (Sparsity)

被修改的像素比例。值越低攻击越稀疏。

稀疏性=修改像素数H×W×C×100%
扰动强度 (Perturbation)

修改的平均幅度，用 ℓ2 范数或 ℓ∞ 范数衡量。

完整参数说明 ​

preprocess 参数 ​

参数类型默认说明
--dataset-rootstrdataset数据集根目录
--datadirstr-数据集名称（必需）
--image-sizeint224目标图像尺寸

train 参数 ​

参数类型默认说明
--backbonestr-骨干网络：resnet50 或 siglip（必需）
--datadirstr-数据集名称（必需）
--epochsint10训练轮数
--batch-sizeint32批大小
--learning-ratefloat1e-4学习率
--devicestrcuda计算设备：cpu 或 cuda
--freeze-siglip-backbone-False冻结 SigLIP 骨干（SigLIP 特有）

attack 参数 ​

参数类型默认说明
--backbonestr-骨干网络（必需）
--checkpointstr-分类器权重路径（必需）
--datadirstr-数据集名称（必需）
--sample-indexint0目标样本索引
--devicestrcuda计算设备

示例 2: 对比两个模型 ​

# 分析结果
echo "Results saved in output/attack/"

快速故障排查 ​

问题解决方案
ModuleNotFoundError检查虚拟环境是否激活
CUDA out of memory减小 --batch-size
数据加载失败检查 --datadir 和数据格式
下载超时设置 export HF_ENDPOINT=https://hf-mirror.com
GPU 未检测到运行 nvidia-smi 检查驱动
