## 🚀 部署说明

### GitHub Pages 配置

1. 进入仓库的 **Settings**
2. 导航到 **Pages** 部分
3. 在 **Source** 中选择:
   - Branch: `main` (或 `doc` 如果使用其他分支)
   - Folder: `/docs`
4. 点击 **Save**

网站会自动部署到 `https://NAC-HUST.github.io/Psoriasis-Robust-Adv-Defense`

### 本地预览

使用 Python 本地服务器测试:

```bash
cd docs
python -m http.server 8000
```

然后访问 `http://localhost:8000`
