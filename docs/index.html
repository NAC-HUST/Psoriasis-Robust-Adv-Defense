项目概览

🎯

研究目标

研究对抗攻击对医学影像分类模型的影响，并提出防御机制，提高诊断的鲁棒性和可信度。

🔍

技术方案

采用 SAMOO 对抗攻击算法，基于多目标优化框架，在保持攻击稀疏性的同时增强攻击效果。

📊

数据支持

支持 CIFAR-10、ImageNette、银屑病数据集，提供完整的预处理和数据加载管道。

🤖

模型架构

集成 ResNet50（CNN）与 SigLIP（Vision-Language）双模态框架，实现 0-shot 和微调两种模式。

关键特性

01

端到端流程

从数据预处理、模型下载，到训练分类器，再到对抗攻击，一键完成全流程。

02

多架构支持

灵活支持 ResNet50 与 SigLIP 骨干网络，可任意组合训练和攻击。

03

稀疏攻击

SAMOO 算法确保对抗样本的稀疏性，更接近实际临床场景中的微小扰动。

快速开始

1

环境配置

git clone https://github.com/NAC-HUST/Psoriasis-Robust-Adv-Defense.git
cd Psoriasis-Robust-Adv-Defense
uv sync
2

预处理数据

uv run main.py preprocess \
  --dataset-root dataset \
  --datadir psoriasis_normal \
  --image-size 224
3

下载模型

uv run main.py download-models
4

训练分类器

uv run main.py train \
  --backbone resnet50 \
  --datadir psoriasis_normal \
  --epochs 3
5

执行攻击

uv run main.py attack \
  --backbone resnet50 \
  --datadir psoriasis_normal

学术输出

研究论文 · 对抗鲁棒性

银屑病医学影像诊断的对抗攻防协同鲁棒性增强方法

本研究提出了一种融合对抗攻击与防御机制的银屑病诊断鲁棒性增强方法， 通过 SAMOO 算法实现稀疏对抗样本生成，并在 ResNet50 与 SigLIP 模型上验证效果。

Cao WJ, Cheng Q, Li YX, Feng ZY, Li YX
